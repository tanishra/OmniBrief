# OmniBrief Frontend

## Local development

```bash
npm install
npm run dev
```

Create a `.env.local` file with:

```env
NEXT_PUBLIC_API_BASE_URL=https://omni.tanish.website
```

## Deploying to Vercel

- Root directory: `frontend`
- Framework preset: `Next.js`
- Environment variable:

```env
NEXT_PUBLIC_API_BASE_URL=https://omni.tanish.website
```
